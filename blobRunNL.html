Blob Run

Dit is een wireframe die ik heb gemaaakt voor mijn game BlobRun.

In mijn eerste jaar moesten wij een ambitie project maken. Sinds ik nog niet wist wat ik wilde doen dacht ik we hebben al een beetje coderen gehad met C# en het maken van simpele spellen. Dus, ik besloot dat ik een 2D platformer wilde maken. Dat is hoe BlobRun geboren was. Ik wilde inspiratie nemen van spellen zoals Mario, dus ik keek voor gratis assets en ben gestart met het maken van de map. Daarna ben ik gestart met het coderen van het springen, lopen en de collisions.

Het gaat over een blob die probeert het einde van zijn avontuur te halen doormiddel van het ontwijken van obstakels. Ik heb alles zelf gemaakt behalve de assets die zijn door Kenney gemaakt. Sinds ik voor dit spel alleen maar een pong kopie in de les had gemaakt moest ik dingen zelf uitvogelen. Maar met de hulp van tutorials en mijn docenten is het mij wel gelukt.

Het is gemaakt met Unity want dat was het programma die we vooral tijdens de les gebruikte. Ik heb ook het internet, en voor de programeer taal heb ik C# gebruikt. Ik heb ook voor het eerst de animator binnen in Unity gebruikt.
