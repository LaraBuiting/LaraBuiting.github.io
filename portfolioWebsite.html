Portfolio website

Voor een school project heb ik voor het eerst geprobeerd om een portfolio website te maken. Sinds ik toen nog in mijn eerste jaar zat had ik nog niet zo veel ervaring met HTML en CSS, maar ik vind dat het nog best goed eruit ziet. Ik heb er veel van geleerd zoals hoe ik mijn spellen speelbaar op de website kon krijgen, door middel van een WebGL build.

Ik heb dit gemaakt om mijn projecten te laten zien op een plek waar mensen het makkelijk bij elkaar kunnen vinden. Ik heb dus geleerd hoe ik spellen embed in mijn spel en hoe ik een contacten pagina maakte.

Programma's die ik heb gebruikt:

Ik heb HTML en CSS voor dit gebruikt via Visual Studio. Ik heb ook Adobe XD gebruikt voor mijn wireframe en Pinterest samen met adobeXD voor mijn inspiratiebord.

Progress images

Eerst heb ik een inspiratiebord gemaakt voor mijn website, ik wilde het kleurrijk maken en daarom heb ik deze kleuren gebruikt want ik vind pastel kleuren heel mooi. Na dat ik de wireframe had gemaakt ben ik gestart om mijn over mij pagina te maken. En als laatste voegde ik mijn spellen toe en de contact pagina.
